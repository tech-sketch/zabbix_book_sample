# Zabbix sample codes

This collection of sample codes has been described in the Zabbix book(ISBN Code: 978-4-7741-6288-1).
[the official site at gihyo.jp](http://gihyo.jp/book/2014/978-4-7741-6288-1?ard=1391386319)
